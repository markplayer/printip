# printip
Takes a modified list of /24 subnets and outputs all ips in those ranges.

inputs a text file containing modified subnets i.e

192.168.1.
192.168.2.
192.168.3.

outputs a text file as shown below:

192.168.1.0
192.168.1.1
192.168.1.2
192.168.1.3
192.168.1.4
192.168.1.5
192.168.1.6
192.168.1.7
192.168.1.8
192.168.1.9
192.168.1.10
192.168.1.11
192.168.1.12
192.168.1.13
